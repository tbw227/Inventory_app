# Frontend — Inventory App

This is a minimal Vite + React frontend scaffold for the Inventory App.

Features included:
- React + Vite setup
- Tailwind CSS (PostCSS)
- Pages: Login, Jobs list
- API helper with axios and token interceptor

Getting started:

```bash
cd frontend
npm install
npm run dev
```

Notes:
- Backend expected at `/api/*`.
- Authentication endpoints are implemented in the frontend; backend JWT endpoints are in `AUTHENTICATION.md` (Phase 2).
- Adjust `package.json` deps if you want to pin versions.
